# 社区网页app使用说明

## 项目依赖

- vue-cli 3.0 
- nodejs10+
- mysql5.7
- 编程软件visio or wenstrom

## Project setup

##### 进入项目

```
cd CommunitySystem
```

##### 安装依赖

```
npm install
```

##### 启动server
```
npm run serve
```

##### 项目打包（非必要）
```
npm run build
```

##### ==注意==要启动两个控制台

- 启动前端项目

```
前面已经提到 yarn/cnpm/npm run server
```

- 启动后端项目

```
cd server
```

- 启动服务

```
node index.js
```
